*Repositorio de desarrollo proyecto TriFix*

En dicho reopsitorio se estara adjuntando el proceso de elaboracion del producto interfaz CRUD y el prototipo que se entregara en el sprint final.

En el desarrollo de este proyecto en el ambito del frontend se usaron herramientas tecnnologicas como:
*Framework: Vite + React.
*Lenguajes de programacion: Javascript, html, css, jsx combinacion de javascript con html propia de REACT.
*Muchas otras dependencias y extensiones como: React-router-dom usado para configurar y administrar la navegacion entre ventanas.
